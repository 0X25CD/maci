# MACI
Minimal Anti-Collusion Infrastructure

This project is based upon https://ethresear.ch/t/minimal-anti-collusion-infrastructure

# Developing MACI

```bash
yarn install

# To compile the circuits
yarn circuit:compile
yarn circuit:setup
yarn circuit:generateverifier
```
